# TFG-disaster-building-footprint-extraction
Sistema automatizado de extracción de huellas de edificios afectados por desastres naturales mediante deep learning y análisis geoespacial. Caso de estudio: erupción volcánica de La Palma (2021). Integra segmentación de imágenes satelitales, regularización geométrica y validación SIG para respuesta ante emergencias.
